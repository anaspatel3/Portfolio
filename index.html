Book Store

Book Store helps user to order the book from number of books from the inventory

The user will be able to see the details of the books.
User can search the book with the help of search bar.
The user can add books to cart, increase number of books, decrease number of books as well as remove from cart.
User can register to the site by providing his username and creating a password.
User can login to the Book Store with the help of username and password.
Form validation is done so that user do not enter the wrong information
Toast notification have been added to notify the user about the action performed by user.
Bill will be calculated automatically based on books in the cart

Technology

Book Store is built using HTML5 and designed by CSS while operations are handled by JAVASCRIPT on top of that add to cart functionality was made with the help of JQuery and the rersponsiveness was made with the help of Boostrap is also used
